## **1.1 미션과 비전**

미션: 카메라를 든 모든 사람이 좋은 사진의 '비결'을 나눌 수 있는 세상을 만든다.

비전: 2027년까지 국내 1위 촬영 정보 공유 플랫폼으로 자리잡고, 글로벌 카메라 커뮤니티로 확장한다.

## **1.2 핵심 가치**

- 신뢰 (Trust): EXIF 없는 사진은 업로드 불가 - 모든 정보는 실제 촬영에 기반
- 재현 가능성 (Reproducibility): 같은 장비·설정으로 누구나 동일 결과물 재현 가능
- 커뮤니티 (Community): 촬영 스팟별 집단 지성 축적
- 개방성 (Openness): 보정 레시피 자유 공유 · 브랜드 중립

# **2\. 문제 정의 및 시장 기회**

## **2.1 문제 정의**

카메라 사용자들은 SNS(인스타그램·플리커 등)에서 멋진 사진을 발견해도, '어디서 찍었는지', '어떤 장비·설정을 사용했는지'를 알기 어렵다. 현재 존재하는 채널의 한계는 다음과 같다.

| **채널**               | **한계점**                            | **결과**              |
| ---------------------- | ------------------------------------- | --------------------- |
| 인스타그램 / 500px     | 장비·설정 정보 미표시, 지도 기능 없음 | 정보 단절 - 재현 불가 |
| 카메라 커뮤니티 게시판 | 텍스트 중심, 위치 정보 없음           | 검색·탐색 비효율      |
| 구글 지도 리뷰         | 사진 위주, 촬영 기술 정보 없음        | 스팟 발견에 그침      |
| 유튜브 / 블로그        | 정보 산재, 비구조화                   | 원하는 스팟 탐색 불가 |

## **2.2 시장 기회**

국내 카메라 시장과 사진 커뮤니티는 꾸준히 성장하고 있으며, 필름 카메라 르네상스·미러리스 대중화로 카메라 입문 인구가 빠르게 증가하고 있다.

| **핵심 시장 지표** | |
| --- | | --- |
| ▶ | 국내 미러리스·DSLR 연간 판매 약 30~40만 대 (KCA 2024 추정) |
| ▶ | DCINSIDE 카메라 갤러리 일 방문자 10만+ / 네이버 사진 카페 회원 150만+ |
| ▶ | 필름 카메라 사용자 MZ세대 중심 연 20% 이상 증가 추세 |
| ▶ | 글로벌 카메라 액세서리·앱 시장 2027년 약 32억 달러 전망 (Grand View Research) |

# **3\. 솔루션 및 제품 개요**

## **3.1 ClickDay란?**

ClickDay는 카메라 사용자가 촬영한 사진을 업로드하면, EXIF 메타데이터를 자동 추출하여 지도 위 핀으로 표시하고, 장비·촬영 설정·보정 레시피를 함께 공유할 수 있는 모바일 + 웹 앱 서비스다.

## **3.2 핵심 기능**

### **① 지도 기반 사진 탐색**

- Google Maps API 연동
- 촬영 위치 중심 클러스터링 핀 표시
- 반경 검색, 지역명 검색, 현재 위치 기반 탐색

### **② EXIF 자동 추출 및 검증 (핵심 차별점)**

- 업로드 시 EXIF 필수 필드(GPS, 카메라 모델, 초점거리, 조리개, 셔터스피드, ISO) 자동 파싱
- EXIF 미포함 사진 업로드 원천 차단 → 정보 신뢰성 보장
- EXIF에서 GPS 좌표 자동 추출 → 지도 핀 자동 배치

### **③ 촬영 정보 카드**

- 카메라 기종 / 렌즈명 / 초점거리
- 조리개(F값) / 셔터스피드 / ISO
- 보정 레시피: 라이트룸 설정, 소니 크리에이티브 룩, 후지필름 필름 레시피 등 (선택 입력)

### **④ 스팟 집단 지성**

- 동일 스팟 다수 사진 축적 → 스팟별 베스트 장비·세팅 자동 요약
- 시간대·계절별 사진 필터링

### **⑤ 소셜 기능**

- 팔로우 / 좋아요 / 저장 / 댓글
- 촬영 스팟 북마크 / 컬렉션

## **3.3 사용자 흐름 (User Flow)**

① 사진 선택 → ② EXIF 자동 추출·검증 → ③ 지도 핀 자동 배치 → ④ 부가 정보 입력(보정 레시피, 태그) → ⑤ 공유 완료

탐색: 지도 탐색 → 핀 클릭 → 사진·정보 카드 열람 → 저장/팔로우

## **3.4 MVP vs. 풀 버전**

| **기능**       | **MVP (2026.05~10)**    | **풀 버전 (2026.11~)** |
| -------------- | ----------------------- | ---------------------- |
| EXIF 파싱·검증 | ✅ GPS·기본 카메라 정보 | ✅ 전체 EXIF 필드      |
| 지도 핀·탐색   | ✅ 기본 클러스터링      | ✅ 고급 필터·반경 검색 |
| 촬영 정보 카드 | ✅ 기본 장비·설정       | ✅ 보정 레시피 입력    |
| 소셜 기능      | ❌ (베타 제외)          | ✅ 팔로우·댓글·저장    |
| 스팟 요약      | ❌                      | ✅ 집단 지성 자동 요약 |
| 수익화 기능    | ❌                      | ✅ 프리미엄·광고       |
